SAE Rushbook

SAE Rushbook is software designed to simplify fraternity rush processes. This software is developed in C# using the .NET Framework. I am currently in the early stages of developing this website, but I hope to have it completed by the end of the school year.

Golf Betting

Golf Betting is an Android application that enables you to place bets on your rounds with friends. The application is built using Java, with a connected database utilizing SQLite. This project is set to be released on the Google Play Store in the coming month.
